# LuccaTi2
Hello there!
Olá, este repositório foi criado para fazer upload do arquivo PNG resultado do desafio "Construindo um esquema conceitual para banco de dados". O projeto foi elaborado tendo em vista
o cenário de uma oficina que realiza consertos e revisões em veículos por meio de equipes de mecânicos e ordens de serviços. As ordens de serviços possuem o valor total do serviço,
além da autorização do cliente, nº da ordem, data de emissão, data estimada para conclusão do serviço, problemas identificados e status atual do serviço. O valor total é calculado
através de consultas em outras tabelas, uma referente ao valor dos serviços e outra referente ao valor das peças necessárias para o conserto/revisão.
Todas as entidades possuem seus próprios relacionamentos para estabelecer uma relação de coesão entre elas e adequação ao contexto.
